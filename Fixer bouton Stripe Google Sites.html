ChatGPT a dit :

Voici les causes possibles et la procédure exacte pour que tes boutons sur Google Sites ouvrent bien un checkout Stripe.

1. Vérifie que tu utilises le bon lien Stripe
Stripe propose deux types de liens :

  • Lien de paiement (Payment Link) → format https://buy.stripe.com/...

  • Lien de checkout hébergé pour un produit → aussi https://buy.stripe.com/...

Dans ton Dashboard Stripe :
Produits → Sélectionne le produit → Créer un lien de paiement → copie le lien.

Si ton bouton Google Sites n’a pas ce lien exact, il n’ouvrira rien.

2. Google Sites n’accepte pas les scripts Stripe
Tu ne peux pas intégrer une vraie page Checkout avec le code JavaScript Stripe (stripe.redirectToCheckout).
Google Sites bloque tout script externe.

Donc : uniquement les liens Stripe directs sont utilisables.

3. Comment connecter le bouton au lien Stripe
Dans Google Sites :

  1. Clique sur ton bouton.

  2. À droite, clique sur Éditer le lien.

  3. Colle le lien Stripe buy.stripe.com.

  4. Valide.

4. Si le bouton n’ouvre toujours rien
Teste ces points :

  • Le lien Stripe fonctionne en ouvrant un nouvel onglet → colle-le dans ton navigateur.

  • Le lien n'est pas tronqué par Google Sites (parfois il coupe après un ?).

  • Le lien ne contient pas d’espace ou de retour à la ligne.

5. Me montrer ton lien
Si tu veux, colle-moi :

  • Le lien Stripe que tu utilises

  • Une capture de la configuration du bouton Google Sites (le texte du lien, pas une image du site)

Je te dis immédiatement ce qui bloque.
